# mortgage-calc

Ипотечный калькулятор с расчётом аннуитетных платежей, переплаты и полного графика погашения. Работает как библиотека Python и как консольная утилита.

---

## Что это делает

Программа рассчитывает параметры ипотечного кредита по аннуитетной схеме — когда вы платите **одинаковую сумму каждый месяц** на протяжении всего срока. В начале срока большая часть платежа идёт на проценты банку, в конце — на погашение основного долга.

Для каждого расчёта программа выдаёт:
- **Ежемесячный платёж** — сколько платить каждый месяц
- **Общую сумму выплат** — сколько денег уйдёт банку за весь срок
- **Переплату** — сколько из этого — чистые проценты
- **График платежей** — помесячная детализация: сколько в конкретном месяце ушло на долг, сколько на проценты, какой остаток долга

Все денежные расчёты ведутся через `decimal.Decimal` с округлением до копеек, чтобы избежать накопления ошибок, характерных для обычных `float`.

## Как пользоваться

### Базовый расчёт

```bash
mortgage --amount 5000000 --rate 7.5 --years 20
```

Вывод:
```
Ежемесячный платёж: 40 280.24 ₽
Общая сумма выплат:  9 667 257.60 ₽
Переплата:           4 667 257.60 ₽
Процент переплаты:   48.28%
```

### Разбор каждого флага

| Флаг | Короткий | Что означает | Пример |
|------|----------|--------------|--------|
| `--amount` | `-a` | Сумма кредита в рублях | `--amount 3000000` |
| `--rate` | `-r` | Годовая процентная ставка в процентах | `--rate 9.5` (9,5% годовых) |
| `--years` | `-y` | Срок кредита в годах | `--years 15` |
| `--format` | `-f` | Формат вывода: `text` (по умолчанию), `json`, `csv` | `--format json` |
| `--schedule` | `-s` | Показать полный график платежей | `--schedule` |
| `--limit` | `-l` | Сколько строк графика показать (по умолчанию 12) | `--limit 5` |
| `--output` | `-o` | Сохранить результат в файл | `--output result.txt` |

### Примеры использования

**С графиком платежей (первые 12 месяцев):**
```bash
mortgage -a 3000000 -r 9.0 -y 15 --schedule
```

**Вывод в JSON (для обработки в другой программе):**
```bash
mortgage -a 1000000 -r 12 -y 1 --format json
```

**Сохранение графика в CSV:**
```bash
mortgage -a 2000000 -r 8.5 -y 10 --format csv --schedule -o schedule.csv
```

**Только сводка, без графика:**
```bash
mortgage --amount 500000 --rate 0 --years 5
```

## Архитектура

```
src/mortgage_calc/
├── __init__.py      # Экспорт публичного API
├── types.py         # Dataclasses: Payment, PaymentSchedule
├── calculator.py    # Ядро: формула аннуитета через Decimal
├── formatter.py     # Форматирование: валюта, таблицы, сводка
└── cli.py           # argparse: точка входа mortgage
```

### Как использовать как библиотеку

```python
from mortgage_calc import calculate

result = calculate(amount=1_000_000, annual_rate=12.0, years=1)

print(f"Платёж: {result.monthly_payment}")
for p in result.schedule[:3]:
    print(f"Месяц {p.month}: {p.total} ₽ (долг {p.principal}, проценты {p.interest})")
```

## Тесты

Покрытие включает:
- Базовые расчёты и краевые случаи (0% ставка)
