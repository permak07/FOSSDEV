# FOSSDEV Projects Repository

Добро пожаловать в единый Git-монорепозиторий для проектов **FOSSDEV**.

## Структура репозитория

Данный репозиторий организован по принципу разделения проектов по Git-веткам. Ветка `main` содержит только общую документацию, а каждый отдельный проект находится и развивается в своей персональной ветке.

### Доступные проекты (ветки)
* **`mortgage_calc`** — Калькулятор ипотеки (разработка ведется в одноименной ветке).
