# Предметная область: ипотечное кредитование

## Аннуитетный платёж

Аннуитет — это равный ежемесячный платёж на весь срок кредита. 
В начале срока большая часть идёт на проценты, в конце — на основной долг.

Формула:

```
P = S × (i × (1 + i)^n) / ((1 + i)^n − 1)
```

Где:
- `P` — ежемесячный платёж
- `S` — сумма кредита
- `i` — месячная процентная ставка (годовая / 12)
- `n` — количество месяцев

## Почему Decimal

Используем `decimal.Decimal` вместо `float` для финансовых расчётов, 
чтобы избежать накопления ошибок округления. Итоговые суммы округляются 
до копеек (2 знака после запятой) по правилу `ROUND_HALF_UP`.

## Особенности реализации

- **Последний платёж**: корректируется так, чтобы остаток был точно 0
- **Нулевая ставка**: деление на 0 не происходит, платёж = сумма / месяцы
- **Hardlinks/симлинки**: не применимо, работаем только с числами

## Валидация

Все входные параметры проверяются:
- Сумма > 0
- Ставка ≥ 0
- Срок > 0 лет
